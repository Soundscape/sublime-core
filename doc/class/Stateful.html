# (void) constructor(states, initialState)

Construct a Stateful object

Examples:

Create a Stateful object 


states =
  engaged:
    greet: (name) ->
      console.log name
      @idle
  idle:
    wait: (instance, deferred) ->
      deferred.resolve()
      @setMachineState @engaged

fsm = new Stateful states, 'idle'

Parameters:

  • states ( Object ) — An object defining the available states and actions
  • initialState ( String ) — The initial state for this instance
