Overview

Container for dependecy catalogs created with Intravenous

Examples:

How to create a Composer 

composer = new Composer()

How to register a class 

composer
  .from 'logging'
  .register 'log', MyLoggerClass

How to register a global value 

composer
  .from 'logging'
  .register 'log', data: 'Hello'

How to instantiate an instance 

log = composer
  .from 'logging'
  .get 'log'
